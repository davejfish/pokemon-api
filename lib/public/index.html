pokemon api

route method description example
/pokemon GET gets a list of all pokemon pokemon
/pokemon/:id GET get the detail about a pokemon with matching id pokemon/1
/pokemon/attacks/:id GET get the attacks for a pokemon with matching id pokemon/attacks/1
/attacks GET get all attacks attacks
/attacks/:id GET get the detail about an attack with matching id attacks/1
/attacks/pokemon-with-attack/:id GET get a list of pokemon with matching attack /attacks/pokemon-with-attack/1
